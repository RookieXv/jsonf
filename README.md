# JSONF

JSONF is a Vue 3 + Vite JSON formatter workstation built with Composition API.

## Features

- Format, minify, validate, and repair JSON
- Preserve escapes mode for API logs and escaped strings
- Two-level toolbar for future tool expansion
- Resizable Input, Output, and Inspector workspace
- JSON tree inspector with JSONPath copy
- Simplified Chinese, Traditional Chinese, and English UI
- Light and dark theme
- Local-only browser processing

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
